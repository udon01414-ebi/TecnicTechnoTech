兄と連絡が取れなくなった。




最後にLINEが来たのは半年前だった。


「就職決まったよ！テクニックテクノテックってところ。
　ちょっと変わった会社だけど、有名なところらしい。
　しばらく研修で忙しくなるけど、落ち着いたら連絡する。」

「配属先の部長、山さんっていう人らしい。
　優しそうな人で、ちょっと安心した。」






それから、既読はつくのに返信がない日が続いた。

会社の採用ページを見ても、代表番号にかけても、
「担当者が不在のため」の一点張り。





最後のメッセージを頼りに、「山さん」に連絡を取ろうとも思った。

会社の採用ページ、組織図、問い合わせフォーム、、
手当たり次第に会社のサイトを探した。





「山」から始まる社員は、何人もいた。

どの人が「山さん」なのか、分からない。

でも、探しているうちに、気づいてしまった。




このサイト、少し変じゃないか？
